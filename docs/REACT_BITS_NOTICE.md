# React Bits UI Notice

The AngeVoice Studio refresh adapts interaction ideas from React Bits:

- Spotlight-style pointer highlights for panels and metrics.
- Glare-style hover feedback for the primary synthesis action.
- Lightweight counter motion for service metrics.

React Bits is published under the MIT + Commons Clause license:
https://github.com/DavidHDev/react-bits/blob/main/LICENSE.md

This project uses those ideas as part of the AngeVoice application UI. It does
not redistribute React Bits as a standalone component library or component
bundle.
